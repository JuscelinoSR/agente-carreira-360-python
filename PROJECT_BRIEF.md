# Project Brief — Agente Carreira 360 em Python

## Problema

Currículos e perfis profissionais ficam desatualizados com facilidade. Além disso, muitos profissionais têm dificuldade em transformar experiências reais em textos claros para recrutadores e redes sociais.

## Solução

Criar um MVP em Python que leia arquivos JSON com dados profissionais e gere automaticamente arquivos em Markdown.

## Funcionalidades iniciais

- Gerar currículo
- Gerar resumo para LinkedIn
- Gerar posts profissionais
- Gerar mensagens de networking
- Analisar vaga de forma simples
- Gerar palavras-chave profissionais

## Escopo

Este projeto é um assistente local. Ele não publica conteúdo automaticamente e não faz integração com redes sociais na fase inicial.

## Critério de sucesso

O usuário deve conseguir executar:

```bash
python mvp/app.py
```

E gerar arquivos dentro da pasta `outputs/`.
