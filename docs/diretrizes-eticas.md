# Diretrizes Éticas

O projeto deve gerar rascunhos e organização profissional, não automações abusivas.

## Regras

- Não inventar experiências.
- Não inventar cursos.
- Não inventar métricas.
- Não publicar automaticamente.
- Não enviar mensagens em massa.
- Não expor dados confidenciais.
- Não prometer contratação.
- Revisar tudo antes de usar publicamente.
