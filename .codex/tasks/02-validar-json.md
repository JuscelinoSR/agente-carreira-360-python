# Tarefa 02 — Validar arquivos JSON

## Objetivo

Revisar os arquivos em `data/` e garantir que todos são JSON válidos.

## Arquivos permitidos

- data/*.json

## Critério de pronto

- Todos os arquivos são JSON válidos.
- Não há dados inventados.
- Campos faltantes devem ficar vazios ou como observação.
