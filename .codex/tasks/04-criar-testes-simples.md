# Tarefa 04 — Criar testes simples

## Objetivo

Criar testes simples para validar se os scripts principais executam.

## Arquivos permitidos

- tests/test_scripts.py

## Regras

- Usar apenas biblioteca padrão.
- Não instalar pytest.
- Pode usar unittest.

## Critério de pronto

- Rodar com `python -m unittest`.
