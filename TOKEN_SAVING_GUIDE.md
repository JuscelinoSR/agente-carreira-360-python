# Guia de Economia de Tokens para Codex

## Use tarefas pequenas

Exemplo:

```text
Codex, execute apenas a tarefa:
.codex/tasks/01-revisar-python-base.md

Não altere outros arquivos.
```

## Evite pedidos amplos

Evite:

```text
Crie tudo, automatize redes sociais, faça site e integre APIs.
```

Prefira:

```text
Crie apenas o script gerar_curriculo.py usando os dados de data/perfil-profissional.json.
```

## Ordem recomendada

1. Revisar estrutura
2. Validar JSON
3. Gerar currículo
4. Gerar LinkedIn
5. Gerar posts
6. Gerar networking
7. Criar menu
8. Testar execução

## Frase para controlar escopo

```text
Se faltar contexto, use [PREENCHER]. Não invente informações.
```
